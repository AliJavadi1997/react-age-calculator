# React Age Calculator

The React Age Calculator is a simple web application designed for practice and study of the useState and useEffect hooks in React. It allows users to input their birthdate and calculates their age in various units, such as years, months, weeks, days, hours, minutes, seconds, and milliseconds.

## Features

- User-Friendly Interface: The application provides a clean and intuitive user interface, making it easy for users to input their birthdate.

- Age Calculation: Utilizing React's useState and useEffect hooks, the application calculates and displays the user's age in different units based on their birthdate.

## Project Structure

The project is organized into the following components:

- InputForm: This component handles user input for the birthdate and triggers the age calculation.

- DisplayResult: This component displays the calculated age in various units.

- AgeCalculator: The main component that integrates InputForm and DisplayResult to create the Age Calculator application.

## Learning Objectives

- Practice with useState and useEffect: Gain hands-on experience with React's useState and useEffect hooks, which are fundamental for managing state and side effects in React applications.

- Component Composition: Learn how to create and compose React components to build a functional application.
